# hdf52json_cm

Convert hdf5 to json functions (auxiliary package for personal use)

## Installation

Get from github

	git clone https://github.com/caiofcm/hdf52json_cm.git

Install with pip (as developer mode (if not remove -e))
	pip install -e .

	python
	import hdf52json_cm
	hdf52json_cm.testing()
